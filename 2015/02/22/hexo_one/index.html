Hexo系列教程之一：使用Hexo+github搭建自己的博客

发表于 2015-02-22 | 分类于 Hexo教程

Hexo [超简单] 从0到1构建属于你的博客

写在前面:
想着要弄一个自己的博客,或记录一些琐碎,或分享一点知识,然后自己在网上查询如何构建自己的博客,通过查询得知用 github + hexo 可以构建一个属于自己的博客,然后自己开始搭建。由于网上的教程太多，不过大多数都差不多，所以自己就把一些最基本的常用的写在这里，也希望帮助一些人简单而快速的构建属于自己的博客。这里是我搭建个人博客的过程记录，希望通过这篇文章能够使更多的人有属于自己的博客。（虽然博客时代已经过去，但是多一个博客也不是不好，对么？）

一、基础博客搭建

1、安装node.js 和 Git

这些需要按照下面的顺序一步步安装，具体也不用知道他们是干什么的，我会持续在博客里简单的介绍这写东西。

1.安装node.js。Node.js中文官网下载到本地，然后安装默认的路径直接安装即可，只需要你一路点击Next。
安装好之后打开命令行（俗称 cmd 键盘快捷键Wind + R）输入 node -v 如果出来所安装的版本号，那恭喜你已经安装成功。

2.安装Git。Git官网 下载文件（直接找 Downloads 按钮即可）然后安装，按照默认路径安装也好，直接选择安装路径也好。如何查看安装成功呢，我们用鼠标右键点击桌面，这是你会看到右键菜单新增两个选择项 “Git GUI Here” 和 “Git Bash Here”。

这里你已经迈出了一小步，下面我们开始迈出一大步。

2、安装Hexo

1.先建一个文件夹，放在那里都可以。例如 :D\hexo
2.进入hexo文件夹，右键点击 Git Bash 。
3.开始安装hexo
输入代码 $ npm install hexo-cli -g
耐心的等待，等待安装完成。
4.初始化 hexo 文件夹
输入代码 $ hexo init
5.安装依赖和插件
$ hexo install
6.启动服务器进行本地预览
$ hexo s
7.当你看到 info hexo is running at http://0.0.0.0:4000/.press ctrl + c stop.时说明你的博客已经安装完成，这时你就可以打开你的浏览器，在地址栏输入 localhost:4000 进行预览。
成功打开已经你就会看到一个博客页面，至此安装就完成了。

3、准备github库来托管我们的博客

1.首先注册github。
2.按照步骤进行注册即可。
3.注册成功以后需要邮箱验证，这里当然是你所注册的邮箱。
4.接下来我们创建一个库。注意username，这里会影响到你的域名！！！所以你必须认真的取个名字。

到这里我们就可以把在本地服务器所看到的网页放到网上了。

hexo命令

`
hexo new ”postName“ #新建文章

hexo generate 生成静态页面至public目录
hexo server 简写hexo s 开始预览访问端口 默认为 4000
hexo deploy 简写hexo d 将deploy目录部署到 github上
